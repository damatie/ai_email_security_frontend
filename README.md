## Overview

This repository contains the frontend code for the AI Email Security Solution. The application provides a dashboard for real-time threat detection, detailed email analysis, and seamless integration with Gmail using color-coded risk labels. It is built with Next.js, React, and TypeScript, following modern development best practices for scalability and maintainability.

## Features

- **Real-Time Threat Detection:** View and analyze threats as emails arrive.
- **Gmail Integration:** Apply risk labels directly within Gmail.
- **User Dashboard:** Detailed metrics, threat distributions, and historical scan data.
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
- **State Management:** Managed with Redux Toolkit for efficient data handling.
- **Code Quality:** Enforced with ESLint, Prettier, and Git hooks (Husky & lint-staged).

## Technologies Used

- **Next.js (v13+):** Utilizing the new App Router for streamlined routing and enhanced server-side capabilities.
- **React:** For building interactive UIs.
- **TypeScript:** For improved type safety and developer experience.
- **Redux Toolkit:** Simplified state management.
- **Tailwind CSS:** Utility-first CSS framework for rapid styling.
- **Husky (v9.1.7) & lint-staged:** For pre-commit Git hooks ensuring code quality.
- **ESLint & Prettier:** Linting and formatting tools.
